# 🎓 College Project

A modern and responsive **College Website** built using **HTML, CSS, and JavaScript**. Designed to provide an intuitive experience for students, faculty, and visitors with a clean UI and smooth navigation.

## ✨ Features
- 📱 Responsive Design
- 🏫 College Information Portal
- 📋 NAAC Navigation Menu
- 📊 Placement & Statistics
- 🎨 Modern UI with Smooth Animations

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript

## 🚀 Getting Started

```bash
git clone https://github.com/Ravindra1807/College-Project.git
```

Open `index.html` in your browser.
